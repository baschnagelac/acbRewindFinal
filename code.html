The Code for APP

                        
                            //get the value/string from the page
                            //controller function
                            function getString() {
                            
                                document.getElementById('alert').classList.add('invisible')
                            
                                let userString = document.getElementById('userString').value;
                            
                                let revString = reverseString(userString);
                            
                                displayString(revString);
                            
                            
                            }
                            
                            //reverse the string 
                            //logic function
                            function reverseString(userString) {
                            
                                let revString = [];
                               
                                
                                //let name = Basch
                                //name[0] = "B";
                                //name[4] = "h";
                                // last position in an array name.length -1
                            
                                //reverse a string using a for loop
                                for (let index = userString.length - 1; index >= 0; index--) {
                                    
                                    revString += userString[index];
                                    
                                }
                            
                                return revString;
                            
                            }
                            
                            //display the reversed string to the user 
                            //view function
                            function displayString(revString) {
                            
                                //write msg to the page
                                document.getElementById('msg').innerHTML = `Your string reversed is:  ${revString} `;
                                //turn on/show the alert box
                                document.getElementById('alert').classList.remove('invisible');
                            
                            }
This code is structured in three functions
function getString()

is the controller function. First, the program begins by making sure that the invisible class remains on the alert ID so that the user doesn't automatically see our messages before the program begins. Next in the controller function is getting the user's input via userString. In this app, we are trying to reverse a string. Therefore we declare that revString is equal to the reversed string of the user's input. And the display of revString is called. function reverseString()

is the logic function. First, the program declares revString as an empty array. Next, a for-loop is used to reverse the user's input string. In order to reverse the string, the program begins at the end of the string and decrements through until the entire string has been reversed. Then, the revString is returned for the following display function.

function displayString()

is the view functino. In the view function, the program grabs the 'msg' Id and uses template literals and innerHTML to display the text "Your string reversed is: ${revString}" with the user's reversed string between the curly brackets. Secondly, the display function also removes the 'invisible' class on the 'alert' ID so the reversed string is revealed to the user while the app runs.
